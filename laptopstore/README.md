# Usage:
1. Clone or download this project to your desktop
2. Create .env.local from example.env:
```sh
cp example.env .env.local
```
3. Install dependencies:
```sh
npm i
```
3. Run project:
```sh
npm start
```

# Build to deploy:
```sh
npm run build
```
